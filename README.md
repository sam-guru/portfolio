## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/sam-guru/portfolio.git
cd portfolio
npm install
```

Run locally:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

Build for production:

```bash
npm run build
```
